# Bitcoin Price Prediction with Linear Regression

This project predicts Bitcoin prices using a linear regression model based on historical data. 
It includes a Python script that uses the scikit-learn library for machine learning and matplotlib for data visualization.

## Screenshot

## Dependencies
- pandas
- scikit-learn
- matplotlib

Install the dependencies using:
```
pip install pandas scikit-learn matplotlib
```
## Usage

1. Clone the repository
2. Run the script
```
python LinearRegression.py
```
The script will train the model on historical data and generate predictions for Bitcoin prices in 2024. 
The results will be displayed in the console, and a plot comparing actual and predicted prices will be saved.

## Results
The script generates predictions for each month in 2024 and displays them along with actual prices in a matplotlib plot.
